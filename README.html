Django REST Framework Web API-ების შექმნის toolkit Django-ში. RESTful API-ების შექმნა Next.js frontend-ისათვის.
Django Channels Django-ს გაფართოება WebSockets-ის ხელშეწყობისთვის. რეალურ დროში ფუნქციების დამატება, მაგალითად, ჩატი.
